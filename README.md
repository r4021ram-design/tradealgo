# tradealgo

A professional, high-performance algo trading platform connecting the Kotak Neo API with a real-time React dashboard.

## Features
- **FastAPI Backend**: Seamlessly validates risk, executes multi-leg options strategies, and tracks live positions using the hardened `OrderManager`.
- **Premium React Dashboard**: Complete with real-time ticks over WebSockets, live Option Chain metrics, a visual Strategy Builder, and an Option Portfolio Manager for scenario analysis.
- **Instrument Master Sync**: Automatically keeps derivative contracts and active expiries in sync.
